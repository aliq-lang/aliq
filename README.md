# ALIQ

Abstract Language Integrated Query for different languages:
- [ALIQ for .Net](https://github.com/sergey-shandar/aliq-dotnet)
- [ALIQ for JavaScript and TypeScript](https://github.com/sergey-shandar/aliq-js)

## Basic Operations

|.Net      |JavaScript|
|----------|----------|
|SelectMany|flatMap   |
|GroupBy   |groupBy   |
|Merge     |merge     |
|Product   |product   |
|Const     |const     |
|Input     |input     |
